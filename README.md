A beautiful, responsive landing page for **TinDog**—the ultimate dating app for dogs! This project was built to practice and showcase modern web design principles, including responsive layouts, navigation bars, and grid systems.

---

## 🚀 Features
* **Responsive Design:** Fully optimized for mobile, tablet, and desktop views using Bootstrap's grid system.
* **Modern UI:** Features a sleek, high-conversion landing page layout.
* **Interactive Components:** Includes a responsive Navbar, Pricing Tables, and a Testimonial Carousel.
* **Custom Styling:** Combines Bootstrap components with custom CSS for a unique brand identity.

---

## 🛠️ Tech Stack
* **HTML5:** Semantic structure for the web page.
* **CSS3:** Custom styling and layout refinements.
* **Bootstrap 4/5:** Framework used for rapid UI development and responsiveness.
* **Google Fonts:** Integrated typography (Montserrat and Ubuntu).
* **Font Awesome:** Iconography for features and social links.
